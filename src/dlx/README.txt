Introduction
----------

The dancing links library (libdlx) uses Donald E. Knuth's Dancing Links
algorithm to solve the exact cover problems. The DLX library which is part of
the Distributed Exact Cover Solver (DECS) suite.


For more information see: http://decs.googlecode.com/
